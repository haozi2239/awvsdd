# Awvsddd
这个小工具实现了一个自动监测漏洞并发送通知到钉钉群组的功能。它使用了AWVS（Acunetix Web Vulnerability Scanner）的API来获取漏洞信息，并通过钉钉的机器人接口实时发送漏洞消息。
